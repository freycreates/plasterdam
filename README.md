# Plasterdam

Open `index.html` through the local preview server, or host this folder as a static website.

Data:

- By default the app loads the official Amsterdam GeoJSON endpoint:
  `https://maps.amsterdam.nl/open_geodata/geojson_lnglat.php?KAARTLAAG=OPENBARE_TOILETTEN&THEMA=openbare_toiletten`

Logo:

- The app uses `assets/logo.png`.
